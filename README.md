# **Elara — Portfolio Intelligence & Net Worth Optimization**

Elara è un’applicazione fintech che unisce **aggregazione del patrimonio**, **stima dei beni fisici**, e **ottimizzazione quantitativa del portafoglio**.  
L’obiettivo è fornire all’investitore un quadro completo del proprio **Net Worth** e proporre strategie ottimali in base al profilo personale, alle preferenze settoriali e alla propria tolleranza al rischio.

---

## Vision

Rendere accessibile a chiunque la gestione avanzata del proprio patrimonio, combinando dati reali, modellazione matematica e machine learning.  
Elara vuole diventare il **copilota finanziario intelligente** per investitori moderni.

---

## Caratteristiche principali

### **1. Aggregazione del portafoglio finanziario**
- Collegamento a broker e banche tramite API (es. Salt Edge / PSD2 quando disponibile).  
- Import di file CSV per broker che non offrono integrazione diretta.  
- Supporto a conti, azioni, ETF, crypto, cash.

### **2. Stima del patrimonio fisico**
- Aggiunta e valutazione di beni fisici (es. elettronica, abbigliamento di valore, oggetti personali).  
- Aggiornamento automatico del valore tramite modelli di *depreciation* e *resell value estimate*.

### **3. Calcolo del Net Worth in tempo reale**
- Dashboard completa del patrimonio.  
- Timeline storica e analisi dell’evoluzione della ricchezza.

### **4. Profilazione dell’investitore**
- Analisi delle preferenze: settori graditi/non graditi.  
- Profilo rischio + orizzonte temporale.  
- Goal finanziari personalizzati.

### **5. Ottimizzazione quantitativa del portafoglio**
- Modelli matematici e metodi moderni:
  - Mean-Variance Optimization  
  - Black-Litterman  
  - Risk Parity  
  - Minimum Variance  
  - Sharpe Ratio Maximization  
- Inclusione di vincoli personalizzati (es. ESG, settori esclusi, limiti di volatilità).

### **6. Consulenza automatizzata (non finanziaria)**
- Simulazioni “what if”.  
- Ribilanciamento consigliato.  
- Alert personalizzati.
